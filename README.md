# Simple_Win_Proxy_Tool
단축키를 지정하여 쉽게 프록시를 켰다 끌 수 있는 도구입니다. 

A tool that allows you to easily turn the proxy on and off by specifying a shortcut key.




1. pip install -r requirements.txt
2. python simple_proxy.py
